Regular wave 30 deg oblique incidence¶

The following sections will be modified in “input.txt” for this case.

Set a descriptive title for your simulation:

!-----TITLE-----
 TITLE = 2D_beach_30deg

WAVEMAKER = WK_REG
DEP_WK = 8.0
Xc_WK = 150.0
Yc_WK = 0.0
Tperiod = 8.0
AMP_WK = 0.5
Theta_WK = 30.0    ! this line
Delta_WK = 3.0
